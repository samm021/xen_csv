# xen_csv

## requirements
  - node v14.9.0
  - npm 6.14.8
## dependecies
  - lodash
  - csv-writer
  - csv-parser
  - csv-write-stream
  
# 
- To install exact packages version
`npm ci`

- To create run then create report & summary : 
`npm start`

- To compile to js : 
`npm prestart`

- To delete results:
`npm run delete:result`
